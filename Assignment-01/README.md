# Assignment 01

This folder contains a simple multi-page conference website for the UET IoT & Robotics Innovation Expo 2026.

## Contents

- Home page with hero section and featured events
- Events page with detailed event cards
- Schedule page for the expo timeline
- Speakers page for featured presenters
- Registration page for attendee sign-up
- Contact page for event inquiries
- Shared stylesheet and image assets

## Structure

- index.html - Main landing page
- events.html - Event listings
- schedule.html - Event timetable
- speakers.html - Speaker information
- register.html - Registration page
- contact.html - Contact information
- style.css - Common styling for the website
- images/ - Folder for all image assets

## Notes

This assignment is designed to practice building a responsive and organized static website using HTML and CSS.
